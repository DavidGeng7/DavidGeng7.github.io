Overview

This tutorial will cover the basics of recursion and recursive functions in Java. Recursion basically makes a method call itself over and over again. This technique can break complicated problems down to simple problems which are easier to solve.

All recursions have a base case and a recursive case. Base cases have no recursive calls, and recursive case calls itself, but usually for a “smaller” task. A recursion method must eventually come to a base case, where recursion stops. In the case that it does not come to a base case, an error will be created due to the recursion running infinitely.


Below is an example of recursion to find the value of base^exponent.

public int exponent (int b, int e)
{
    if(e == 0)
        return 1;
    return b * exponent (b, e - 1);
}

The base case here is e == 0 where we return 1 as b^0 is always 1. In the case where b = 2 and e = 3, the recursion basically works like this:

  • 2 * exponent(2, 2)
  • 2 * (2 * exponent (2, 1))
  • 2 * (2 * (2 * exponent (2, 0))
  • 2 * (2 * (2 * (1))) = 8


Although recursion can solve complicated problems easily, and have many implementations, it can be hard to understand for people reading the program. It can also be hard to come up with a recursive function to perform a specific task, which is why some prefer iterative functions over recursive functions.


In summary, recursion is the act of a method calling itself. It must have a recursive case and base case. And it can be hard to understand, so take your time!
