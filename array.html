Overview

This tutorial will teach you about the basics of the Array data structure in Java. Arrays are extremely useful data stucture in many situations due to its ability to hold a large amount of value of the same data type.

Arrays have a fixed size, meaning once you create an array object, its length cannot be modified.

Each value inside an array is called an element, which can be accessed by its unique index. The index of an array starts at 0, so the 1st element is accociated with index 0, 2nd with index 1, 3rd with index 2, and so on.

You can declare and initialize an array by using the following syntax: 

int [] arr1 = new int[3];
arr1[0] = 0;
arr1[1] = 1;
arr1[2] = 2;

int [] arr2 = {0, 1, 2};

The general rule is as follows:

dataType [] identifier = new dataType [size];

You can access a supecific element by using identifier[index], or access all elements of an array once using a for loop: 

int [] arr = {1, 3, 5, 7, 9, 11};
System.out.println(arr[0]) //Prints 1
for(int i = 0; i < arr.length; i++)
    System.out.println("Element at index " + i + " is " + arr[i]);

2D Arrays

An array can be multi-dimensional, which are arrays of arrays. The number of square brackets indicates the dimensions in the multi-dimentsional array. In the case of a 2D array, there are two square brackets indicating row and col number.

You can declare and initialize an 2D array by using the following syntax: 
int [][] arr2D1 = new int [2][2];
arr2D1[0][0] = 1;

int [][] arr2D2 = {{1, 2, 3}, {4, 5, 6}};

To access a specific element of a 2D array, you need to specify the index for each of the square brackets or traverse it using a nested for loop to access all elements in the 2D array once. 

int [][] arr = {{1, 2, 3, 4,}, {5, 6, 7, 8}};
int element = arr[0][0] //element = 1
for(int row = 0; row < arr.length; row ++){
    for(int col = 0; col< arr.length; col ++)
        System.out.println("Element at row " + row + " and col " + col " is " + arr[row][col]);
}

One issue with Array is that its size cannot be modified once it is declared, which is why Java offers an alternative class called ArrayList which you can learn in the ArrayList tutorial.
