Overview

This tutorial will teach you the basics of the ArrayList data structure in Java. ArrayList is basically Arrays with the ability to change size. It can be considered an upgraded version of Array. Although its downside is that it can take up a lot of memory space due to its flexibility.

To create an ArrayList, you must first import the Java utilities package using import java.util.*; as ArrayList is not built-in. The data type of an ArrayList must always be a class instead of a primitive, i.e. Integer, String, etc.

The syntax to create an ArrayList is as follows: 

ArrayList <Integer> arrlist = new ArrayList <Integer>();

There are only a couple of important methods to manipulate an ArrayList that you should know and remember, the other methods associated with the class can be found in the Java documentation. These methods include: size(), add(), set(), get(), remove().

Usage are as follows:

  • add() adds an element to the arraylist
  • size() gets the size of the arraylist
  • set() sets a specific index of the arraylist to a new element
  • remove() removes the element at a specific index
  • get() gets the element at a specific index

ArrayList <Integer> arrlist = new ArrayList <Integer>();
arrlist.add(7);
arrlist.add(17);
arrlist.add(77);
System.out.println(arrlist.get(2)); //print 77
System.out.println(arrlist.size()); //print 3
arrlist.remove(0) //removes 7
arrlist.set(0, 4) //change 17 to 4

As discussed before, although flexibility is great, ArrayList uses more memory space and is generally slower than Array, which makes Arrays the better pick most of the times. One should only use ArrayList when the size changeability of the ArrayList is needed.
